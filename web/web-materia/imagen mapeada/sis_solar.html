Definición y uso


El atributo usemap especifica una imagen como un
mapa de imagen del lado del cliente (un mapa de imagen es una imagen
con áreas en las que se puede hacer clic).
El atributo usemap está asociado con el atributo de nombre de un elemento
<map> y crea una relación entre <img> y <map>.

Nota: El atributo usemap no se puede utilizar si el elemento <img> es descendiente
de un elemento <a> o <button>.
