Conceptos basicos

Qué es internet?

El nombre Internet procede de las palabras en b inglés Interconnected Networks , que significa “redes interconectadas”.

Qué es ARPAnet?

es el precedente directo de internet

Qué es web?

Una página web, pagina electronica o ciberpágina, es un documento o informacion electronica capaz de contener texto

Qué es Hipertexto?

aquel texto que contiene enlaces a otros documentos

Qué es LINK?

Un enlace o link es texto o imágenes en un sitio web que un usuario puede pinchar

Qué es www?

WWW son las iniciales que identifican a la expresión inglesa

Qué es HTTP?

el HTTP son las siglas de “Hypertext Transfer Protocol” es un protocolo de transferencia donde se utiliza un sistema mediante el cual se permite la transferencia de información
